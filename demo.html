diagrams-ts Examples

Draw cloud system architecture diagrams as code in TypeScript

Example 1: Simple Flow

TypeScript Code
import { Diagram, Node } from "diagrams-ts";

const diagram = new Diagram("Simple Flow", {
  direction: "TB",
  outformat: "svg",
});

const web = diagram.add(new Node("Web"));
const app = diagram.add(new Node("App"));
const db = diagram.add(new Node("DB"));

web.to(app).to(db);

const svg = await diagram.render();
diagram.destroy();

Example 2: AWS Architecture

TypeScript Code
import { Diagram } from "diagrams-ts";
import { EC2 } from "diagrams-ts/providers/aws/compute";
import { RDS } from "diagrams-ts/providers/aws/database";

const diagram = new Diagram("AWS", {
  direction: "TB",
});

const web = diagram.add(new EC2("Web"));
const db = diagram.add("new RDS("DB"));

web.to(db);

const svg = await diagram.render();
