# Providers

ts-diagrams includes 17 providers with over 1000 node classes.

## Available Providers

| Provider          | Import Path                  | Services                     |
| ----------------- | ---------------------------- | ---------------------------- |
| **AWS**           | `ts-diagrams/aws/*`          | 500+ services                |
| **Azure**         | `ts-diagrams/azure/*`        | 400+ services                |
| **GCP**           | `ts-diagrams/gcp/*`          | 100+ services                |
| **Kubernetes**    | `ts-diagrams/k8s/*`          | Pods, Deployments, Services  |
| **Alibaba Cloud** | `ts-diagrams/alibabacloud/*` | Compute, Database, Network   |
| **DigitalOcean**  | `ts-diagrams/digitalocean/*` | Compute, Database, Network   |
| **Elastic**       | `ts-diagrams/elastic/*`      | Elasticsearch, Kibana, Beats |
| **Firebase**      | `ts-diagrams/firebase/*`     | Develop, Grow, Quality       |
| **Generic**       | `ts-diagrams/generic/*`      | VPN, OS, Storage             |
| **GIS**           | `ts-diagrams/gis/*`          | Data, Desktop, Mobile        |
| **IBM**           | `ts-diagrams/ibm/*`          | Compute, Data, Network       |
| **OCI**           | `ts-diagrams/oci/*`          | Oracle Cloud Infrastructure  |
| **On-Premises**   | `ts-diagrams/onprem/*`       | Docker, Kubernetes, DBs      |
| **OpenStack**     | `ts-diagrams/openstack/*`    | Compute, Networking          |
| **Outscale**      | `ts-diagrams/outscale/*`     | Compute, Network             |
| **Programming**   | `ts-diagrams/programming/*`  | Languages, Frameworks        |
| **SaaS**          | `ts-diagrams/saas/*`         | Third-party services         |

## Usage

```typescript
import { EC2, Lambda } from "ts-diagrams/aws/compute";
import { RDS } from "ts-diagrams/aws/database";
import { GCE } from "ts-diagrams/gcp/compute";
```

## Provider Structure

Each provider is organized by service category:

```
providers/aws/
  ├── compute.ts     # EC2, Lambda, ECS, etc.
  ├── database.ts    # RDS, DynamoDB, etc.
  ├── storage.ts     # S3, EBS, etc.
  └── ...
```
