# Basic Diagrams

## Creating a Diagram

```typescript
import { Diagram } from "ts-diagrams";

const diagram = new Diagram("My Diagram", {
  direction: "LR", // Left to Right (default)
  // direction: 'TB' // Top to Bottom
});
```

## Adding Nodes

```typescript
import { EC2 } from "ts-diagrams/aws/compute";

const server = diagram.add(new EC2("Web Server"));
```

## Connecting Nodes

### Single Connection

```typescript
web.to(database);
```

### Chaining Connections

```typescript
client.to(web).to(api).to(database);
```

### Multiple Targets

```typescript
web.to([api1, api2, api3]);
```

### Bidirectional Connections

```typescript
web.to(database);
database.to(web); // Or use .with()
```

## Rendering

```typescript
// Get SVG string
const svg = await diagram.render();

// Save to file
await diagram.save("output.svg");

// Render as PNG
const png = await diagram.render({ format: "png" });
```

## Diagram Options

```typescript
const diagram = new Diagram("My Diagram", {
  name: "my-diagram",
  direction: "TB",
  curvestyle: "ortho", // ortho, curved, spline, polyline
  theme: "neutral", // neutral, dark, forest, mint, ocean, sky
  autolabel: false,
});
```
