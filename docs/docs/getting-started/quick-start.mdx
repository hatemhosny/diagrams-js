# Quick Start

Let's create your first diagram!

## Basic Diagram

```typescript
import { Diagram } from "ts-diagrams";
import { EC2 } from "ts-diagrams/aws/compute";

const diagram = new Diagram("My First Diagram");
const server = diagram.add(new EC2("Web Server"));

const svg = await diagram.render();
console.log(svg);
```

## Connecting Nodes

```typescript
import { Diagram } from "ts-diagrams";
import { EC2 } from "ts-diagrams/aws/compute";
import { RDS } from "ts-diagrams/aws/database";

const diagram = new Diagram("Web Architecture");

const web = diagram.add(new EC2("Web Server"));
const db = diagram.add(new RDS("Database"));

// Create connection
web.to(db);

const svg = await diagram.render();
```

## Multiple Connections

```typescript
import { Diagram } from "ts-diagrams";
import { EC2 } from "ts-diagrams/aws/compute";
import { RDS } from "ts-diagrams/aws/database";
import { S3 } from "ts-diagrams/aws/storage";

const diagram = new Diagram("Full Stack");

const web = diagram.add(new EC2("Web"));
const api = diagram.add(new EC2("API"));
const db = diagram.add(new RDS("DB"));
const storage = diagram.add(new S3("Storage"));

// Chain connections
web.to(api).to(db);
api.to(storage);

const svg = await diagram.render();
```

## Save to File

```typescript
const diagram = new Diagram("My Diagram");
// ... add nodes and connections

await diagram.save("my-diagram.svg");
```
